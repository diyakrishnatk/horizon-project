## 📌 A Full-stack Tour Management Web Application

Horizon is a web application developed using MERN technologies. It helps tour operators manage and organize tours efficiently.

The application includes features such as:
- Tour package creation  
- Itinerary customization  
- Pricing management  
- User authentication  
- Search and filtering  
- Review and rating system  

Users can explore and book tours based on their preferences, making travel planning easier and more efficient.
